# TaxAlly

## Project Overview
TaxAlly is a real-time, intelligent tax & compliance assistant designed to remove fear, reduce dependency on intermediaries, and enable micro-businesses to become formally compliant with confidence. It understands the userâ€™s business context, tracks obligations automatically, explains actions in simple language, and prevents non-compliance before it occurs.

## Key Features
- **Business Profiling:** Tailored compliance based on nature, turnover, and geography.
- **Real-Time Compliance Calendar:** Tracks GST, IT, TDS, and local laws.
- **Plain-English Explanations:** Zero jargon mode for better understanding.
- **Automated Alerts:** Risk warnings before deadlines.
- **Conversational Copilot:** Chat-based guidance for instant help.

## Setup and Installation Instructions

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd GFGBQ-Team-Hawks-2
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```

## Usage Instructions
1.  **Dashboard:** Upon launching, you will see the main dashboard with an overview of your compliance status.
2.  **Copilot:** Navigate to the "Copilot" tab to ask questions and get instant, jargon-free answers about tax laws.
3.  **Reports:** Use the "Reports" section to analyze expenses and generate compliance reports.
4.  **Invoices:** Manage your invoices and track payments.
